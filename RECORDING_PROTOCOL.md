# بروتوكول تصوير الوصولة بعد السكتة الدماغية
## Stroke Reach-to-Target Video Recording Protocol

---

## الهدف
تسجيل فيديوهات موحدة يمكن مقارنتها باستخدام SPARC ومتغيرات الوصولة.

## 1. إعداد الكاميرا
- **التليفون ثابت تمامًا** على ترايبود أو حامل. لا حد يحركه.
- **الارتفاع**: مستوى الكتف تقريبًا.
- **المسافة**: 2–2.5 متر من المريض.
- **الزاوية**: جانبية (sagittal) لليد المتحركة. لو مش ممكن، استخدم نفس الزاوية لكل المرضى.
- **الإضاءة**: قوية ومنتشرة، بدون ظلال على الجسم.

## 2. إعداد المريض
- يقف أو يجلس على كرسي بدون مسند ذراعين.
- **اليد البداية**: على الفخذ أو على طاولة أمامه، ثابتة.
- **الهدف**: كوب أو زجاجة على طاولة أمامه مباشرة.
- **مسافة الهدف**: 30 سم أمام اليد البداية.

## 3. المهمة
1. المريض يرفع يده من البداية إلى الهدف.
2. يلمس الهدف.
3. يرجع يده للمكان البداية.
4. **مهمة واحدة فقط** في كل فيديو.

## 4. مدة الفيديو
- **3–5 ثواني فقط**.
- لو الفيديو أطول، قصه قبل التحليل.

## 5. التسجيلات المطلوبة لكل مريض
- يد مصابة (affected)
- يد سليمة (healthy) — نفس الإعداد بالضبط

## 6. قبل التحليل
- حدد الجانب: left أو right
- حدد زاوية الكاميرا: frontal / oblique / sagittal
- تأكد إن اليد فعلاً تحركت مسافة مناسبة

## 7. أمر التحليل
```bash
python analyze_video.py --video patient_pre.mov --side right --view sagittal --label pre --out report.json
```

## 8. متى النتيجة تكون صالحة للمقارنة؟
- reach_amplitude_sw ≥ 0.30
- movement_time_sec بين 0.5 و 3.0
- نفس زاوية الكاميرا للتسجيلين
- نفس مسافة الهدف
